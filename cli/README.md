# career-cli

> 基于 umijs 封装的脚手架工具

## Getting Started

Install, create and start.

```bash
# Install
$ npm install career-cli -g

# Create app
$ jk new myapp

# Start app
$ cd myapp
$ npm install
$ npm start
# yarn 同上
```

## Commands

### jk new <appName> [options]

Create app with new directory.

#### Usage Examples

```bash
$ jk new myapp
```

## 打包

```bash
# 普通打包
$ npm run build
```
